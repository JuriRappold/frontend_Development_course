The style attribute

The style attribute uses CSS properties (more later), like this style="property1:value1; property2:value2; ...", the properties are separated by semicolons.

CSS properties & values

Each bullet point show cases the property of style.
  • "color" is for text-color
  • "background-color" for the background of an element
  • "font-family" is the font-type
  • "font-size" decides the size, one can use a variety of methods (percentages, px, and others)
  • "text-align" aligns text, does it work with bullet points?

Links/Anchor Tags

Any HyperLink works with the anchor-tag (a). The two most important attributes are href="" and target="". href is what it links to, target defines how the link is opened (same window, new window, new tab). _blank opens in new window

One can also use links within the internal html document to link to other sections. For that to work, the linked element has to have a non-null attribute id="". A link to an html element references it like so: href="#element_id". This link goes to the top of this page
